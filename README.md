# React Todo App

React와 Vite를 사용하여 만든 간단한 Todo List 애플리케이션입니다.

## 📝 주요 기능

- 할 일 추가
- 할 일 삭제
- 완료 체크 (토글)
- localStorage를 활용한 데이터 유지
- 새로고침 시 기존 데이터 자동 로드

## 🧩 컴포넌트 구조

- `TodoForm.jsx` : 입력 및 추가 기능
- `TodoList.jsx` : Todo 목록 렌더링
- `TodoItem.jsx` : 개별 Todo 항목
- `useTodos.js` : 상태 및 로직 관리 (custom hook)
- `storage.js` : localStorage 유틸 함수

## 🎨 UI

- TailwindCSS를 사용하여 간단한 스타일 적용
- 미니멀한 디자인과 직관적인 UX 구현
